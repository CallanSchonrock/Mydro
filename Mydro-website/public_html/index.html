What is Mydro?

Mydro is a Hydrologic Rainfall Runoff Routing Software designed to simulate catchment flows from rainfall. With a database manager similar to TUFLOW, event and scenario management becomes a trivial task. Mydro outputs csv files for each simulation, for each subcatchment, for both routed and local flows.
Additional functionality is provided for optional rating curves, where a database of rating curves can be used to provide water level estimates at specified subcatchment outlets. QMydro is the official subcatchment manager, providing outputs for Mydro's necessary catchment characteristics of:

  • Subcatchment Area
  • Main Channel Length
  • Channel Slope
  • Characteristic Hill Length
  • Characteristic Hill Slope
